Mitutoyo USB-ITN Digimatic interface¶

This page describes the API (programmers interface) for this instrument. Information on how to connect to this instrument is given in the documentation of the instruments module

class xlab.instruments.manufacturers.mitutoyo.usb_itn.USB_ITN(port)[source]¶

Interface for the USB-ITN USB-to-Digimatic (Mitutoyo) interface. Requires driver to be installed; see the USB-ITPAK User’s manual section 2.2

read()[source]¶

Reads the Digimatic sensor. Returns float.

Raises TimeoutError upon communication timeout (e.g. cable not connected) or IOError upon reception of an invalid response
